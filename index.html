Niveau vs direction
Calls / Puts — le niveau ne compte pas
Ton compte bancaire est à −300€. Tu reçois +200€. Il monte à −100€.
Tu es encore dans le rouge mais tu reçois de l'argent. C'est pareil pour les lignes calls/puts.

Ligne calls à −200M qui monte vers −100M = MMs achètent spot. Peu importe le niveau.
Ligne calls à +200M qui descend vers +100M = MMs vendent spot. Peu importe le niveau.

Notional 1H total — le niveau ET la direction comptent
Le niveau = biais accumulé depuis le début de la session
La direction = ce qui se passe maintenant
Au-dessus de 0 + monte ↑
Biais haussier qui s'amplifie.
Au-dessus de 0 + descend ↓
Biais haussier qui s'érode.
En-dessous de 0 + descend ↓
Biais baissier qui s'amplifie.
En-dessous de 0 + monte ↑
Biais baissier qui s'allège.
Régime gamma
GAMMA + — amortisseur
MMs font l'inverse du marché. Moves étouffés.
HIRO haussierHausse limitée, fade possible
HIRO baissierBaisse amortie, rebond probable
GAMMA − — accélérateur
MMs font la même chose que le marché. Moves amplifiés.
HIRO haussierRally amplifié, laisser courir
HIRO baissierSell-off qui s'emballe
